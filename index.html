Projects

Uber Data Analysis

Using R programming, I conducted an Uber data analysis to gain insights into customer behavior. I started by reading the data and extracting time information from datetime fields using the lubridate package. The data was then tidied with dplyr and tidyr, ensuring it was ready for analysis. For visualization, I used ggplot2 to create insightful plots, enhanced with ggthemes for better themes and scales. I grouped the data by month and day to understand trip patterns over time. The scales package helped map the data to the correct scales, and DT was used to create interactive datatables that made it easy to explore the data further. This comprehensive approach provided a clear understanding of the customers who avail Uber trips.
